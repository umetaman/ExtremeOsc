### Unity Package Managerからインストール

Package Managerの [+] > Install from git URL... から `https://github.com/umetaman/ExtremeOsc.git?path=Assets/ExtremeOsc` を追加してください。

```
https://github.com/umetaman/ExtremeOsc.git?path=Assets/ExtremeOsc
```


### .unitypackageからインストール

GitHubの[Releases](https://github.com/umetaman/ExtremeOsc/releases/)ページから.unitypackageをダウンロードして、Unityのプロジェクトにインポートしてください。
